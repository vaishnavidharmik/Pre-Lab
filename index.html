HTML Notes

Header Tags

tag h1 is used for the main heading of the page.

tag h2 is used for subheadings.

tag h3 is used for inner contain.

tag h4 is used for inner contain.

tag h5 is used for inner contain.

tag h6 is used for inner contain.

Text Styling

Bold text can be achieved using the b tag.

Italic text can be achieved using the i tag.

Underlined text can be achieved using the u tag.

Setting Text Color

To set text color, you can use the CSS color property. For example:

h1{
color:#333;
}

This will set the color of all h1 tags to a dark gray (#333).
